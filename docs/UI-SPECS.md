
**Font Weights:**

## Components Library

### Button

**Variants:**
- Primary: Blue background, white text
- Secondary: Gray background, gray-900 text
- Outline: White background, border, primary text
- Ghost: No background, primary text on hover
- Destructive: Red background, white text

**Sizes:**
- sm: 32px height, 12px padding
- md: 40px height, 16px padding (default)
- lg: 48px height, 20px padding

**States:**
- Default
- Hover: Darker shade
- Active: Even darker
- Disabled: Gray, cursor not-allowed
- Loading: Spinner icon

### Input

**Types:**
- Text
- Number
- Email
- Password
- Textarea
- Select
- Date
- Time

**States:**
- Default
- Focus: Blue ring
- Error: Red border, error message below
- Disabled: Gray background

### Badge

**Variants:**
- Default: Gray
- Success: Green
- Warning: Yellow
- Error: Red
- Info: Blue

**Example:**
```tsx
<Badge variant="success">Active</Badge>
<Badge variant="warning">Planning</Badge>
<Badge variant="error">Cancelled</Badge>
```

### Card

**Structure:**
```tsx
<Card>
  <CardHeader>
    <CardTitle>Title</CardTitle>
    <CardDescription>Description</CardDescription>
  </CardHeader>
  <CardContent>
    Content here
  </CardContent>
  <CardFooter>
    <Button>Action</Button>
  </CardFooter>
</Card>
```

### Table

**Features:**
- Sortable columns
- Pagination
- Row selection
- Expandable rows
- Fixed header on scroll

### Dialog/Modal

**Sizes:**
- sm: 400px
- md: 600px (default)
- lg: 800px
- xl: 1000px
- full: 95vw

**Structure:**
```tsx
<Dialog>
  <DialogTrigger>Open</DialogTrigger>
  <DialogContent>
    <DialogHeader>
      <DialogTitle>Title</DialogTitle>
      <DialogDescription>Description</DialogDescription>
    </DialogHeader>
    <div>Content</div>
    <DialogFooter>
      <Button variant="outline">Cancel</Button>
      <Button>Save</Button>
    </DialogFooter>
  </DialogContent>
</Dialog>
```

### Toast Notification

**Position:** Bottom-right

**Variants:**
- Success: Green icon + border
- Error: Red icon + border
- Warning: Yellow icon + border
- Info: Blue icon + border

**Duration:** 5 seconds (configurable)

**Actions:** Dismiss, optional action button

## Responsive Design

### Breakpoints

```css
--screen-sm: 640px;
--screen-md: 768px;
--screen-lg: 1024px;
--screen-xl: 1280px;
--screen-2xl: 1536px;
```

### Mobile Adaptations

**< 768px (Mobile):**
- Sidebar collapses to hamburger menu
- Stats cards stack vertically
- Calendar switches to day view by default
- Tables become card lists
- Multi-column forms become single column
- Modals take full screen

**768px - 1024px (Tablet):**
- Sidebar remains visible but collapsible
- Stats cards in 2 columns
- Calendar shows 3-4 days
- Tables remain tables

**> 1024px (Desktop):**
- Full layout
- All features visible

## Print Styles

**Print-specific CSS:**
```css
@media print {
  /* Hide non-essential elements */
  .no-print { display: none; }

  /* Adjust colors for B&W printing */
  * {
    color: black !important;
    background: white !important;
  }

  /* Page breaks */
  .page-break {
    page-break-after: always;
  }

  /* Ensure proper sizing */
  body {
    font-size: 12pt;
  }
}
```

**Print button triggers:**
- Browser print dialog
- Optional: PDF generation via API
